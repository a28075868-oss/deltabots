# 🌐 Web-панель управления Minecraft ботами

Веб-интерфейс для удобного управления ботами через браузер!

## 🚀 Быстрый старт

### 1. Установите зависимости (если еще не установлены):
```bash
npm install
```

### 2. Запустите веб-сервер:
```bash
npm run web
```

### 3. Откройте в браузере:
```
http://localhost:3000
```

## ✨ Возможности веб-панели

### ⚙️ Настройки сервера
- Адрес сервера (домен или IP)
- Порт подключения
- Версия Minecraft
- Пароль для автоматического логина

### 🎮 Настройки ботов
- Количество ботов (1-50)
- Префикс имени ботов
- Задержка между спавном ботов
- Задержка переподключения

### 🎯 Опции поведения
- ✅ Автоматическое возрождение
- ✅ Случайное движение
- ✅ Ответы в чате
- ✅ Автоматический логин

### 🎛️ Управление
- **Запустить ботов** - запуск всех ботов одновременно
- **Остановить всех** - остановка всех активных ботов
- **Обновить список** - обновление информации о ботах

### 📋 Мониторинг ботов
Для каждого бота показывается:
- 🤖 Имя бота
- ✅ Статус (Онлайн/Подключается)
- ❤️ Здоровье
- 🍖 Уровень голода
- 📍 Координаты (X, Y, Z)

### 📝 Логи
- Просмотр логов каждого бота отдельно
- Все события: подключение, команды, сообщения в чате
- Автоматическое обновление

### 🎮 Индивидуальное управление
- Отправка команд конкретному боту
- Остановка конкретного бота

## 🎨 Типы поведения ботов

При запуске боты автоматически распределяются по типам:

1. **🏃 Бегун** - активно бегает по серверу
2. **🎮 Командир** - использует команды `/rtp`, `/spawn`, `/warp`
3. **💬 Болтун** - пишет в чат и отвечает игрокам

## 🔧 Технические детали

### Backend (server.js)
- **Express.js** - веб-сервер
- **REST API** - для управления ботами
- **Mineflayer** - подключение к Minecraft

### Frontend (public/)
- **HTML5** - структура
- **CSS3** - стили и анимации
- **JavaScript** - логика и API запросы

### API Endpoints

```
GET  /api/config          - Получить конфигурацию
POST /api/config          - Сохранить конфигурацию
GET  /api/bots            - Список активных ботов
POST /api/start           - Запустить ботов
POST /api/stop            - Остановить всех ботов
POST /api/stop/:botName   - Остановить бота
POST /api/command/:botName - Отправить команду боту
GET  /api/logs/:botName   - Получить логи бота
```

## 🎯 Автоматические команды

Каждый бот автоматически выполняет:
1. `/login qwer123Q` (через 1 сек)
2. `/kit start` (через 2.5 сек)
3. `/rtp safe` (через 4 сек)

## 💡 Советы

1. **Сначала настройте** - укажите правильный адрес сервера и пароль
2. **Сохраните** - нажмите "Сохранить настройки"
3. **Запустите** - нажмите "Запустить ботов"
4. **Мониторьте** - список обновляется автоматически каждые 5 секунд

## ⚠️ Важно

- Веб-панель работает только локально (localhost)
- Не закрывайте терминал с запущенным сервером
- Для доступа из сети нужна дополнительная настройка

## 🔌 Порт

По умолчанию веб-сервер запускается на порту **3000**.
Если порт занят, измените в `server.js`:
```javascript
const PORT = 3000; // Измените на другой
```

## 📞 Помощь

Если возникли проблемы:
1. Проверьте, что `npm install` завершился успешно
2. Проверьте, что порт 3000 свободен
3. Откройте консоль браузера (F12) для деталей ошибок
4. Проверьте логи в терминале
