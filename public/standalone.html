📦 Установка и запуск

1. Установите Node.js

Скачайте с nodejs.org

2. Скачайте файлы проекта

Клонируйте репозиторий или скачайте ZIP архив

3. Установите зависимости

npm install

4. Экспортируйте config.json

Нажмите кнопку "Экспорт config.json" выше

5. Запустите ботов

npm run client

⚠️ Важно

  • Vercel не поддерживает долгие WebSocket соединения
  • Боты должны работать на вашем компьютере или VPS
  • Эта веб-панель только для настройки конфигурации
  • Для постоянной работы используйте Railway, Render или VPS
