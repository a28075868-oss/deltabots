# Minecraft Боты для DeltaWorld

Боты для автоматического подключения к Minecraft серверу DeltaWorld с разными типами поведения.

## Информация о сервере

- **Домен:** mc.deltaworld.go-mc.pro
- **IP:** 166.1.144.20
- **Порт:** 25565
- **Пароль:** qwer123Q (автоматически вводится)

## Установка

1. Убедитесь, что у вас установлен Node.js (версия 16 или новее)
2. Установите зависимости:

```bash
npm install
```

## Использование

### Запуск нескольких ботов с разным поведением

```bash
npm start
```

Это запустит ботов, которые будут вести себя по-разному:
- 🏃 **Бегуны** - бегают во все стороны, прыгают, используют спринт
- 🎮 **Командиры** - используют команды `/rtp safe`, `/kit start`, `/spawn` и другие
- 💬 **Болтуны** - пишут в чат, общаются с игроками, отвечают на сообщения

### Запуск одного бота

```bash
npm run bot
```

### Запуск умного бота с командами

```bash
npm run smart
```

## Типы поведения ботов

### 🏃 Бот-Бегун (Runner)
- Бегает в случайных направлениях
- Использует спринт
- Прыгает во время бега
- Движется во все стороны: вперед, назад, влево, вправо

### 🎮 Бот-Командир (Commander)
- Использует серверные команды:
  - `/rtp safe` - случайная телепортация
  - `/kit start` - стартовый набор
  - `/spawn` - телепорт на спавн
  - `/home` - телепорт домой
  - `/warp` - телепорт на варп
- Выполняет команды каждые 30-90 секунд
- Сразу после входа использует `/kit start`

### 💬 Бот-Болтун (Chatter)
- Пишет в чат сообщения:
  - "Привет всем!"
  - "Как дела?"
  - "Что делаете?"
  - "Приятной игры!"
  - И другие...
- Отвечает на сообщения игроков
- Реагирует на приветствия и вопросы
- Пишет каждые 45-120 секунд

## Настройка

Отредактируйте файл `config.json`:

```json
{
  "server": {
    "host": "mc.deltaworld.go-mc.pro",
    "port": 25565,
    "version": "1.16.5"
  },
  "bots": {
    "count": 5,              // Количество ботов
    "namePrefix": "test",    // Префикс имени (test1, test2, и т.д.)
    "reconnectDelay": 5000,  // Задержка переподключения (мс)
    "spawnDelay": 2000       // Задержка между запуском ботов (мс)
  },
  "behavior": {
    "autoRespawn": true,      // Автоматическое возрождение
    "randomMovement": true,   // Случайное движение
    "chatResponses": true     // Ответы в чате
  },
  "auth": {
    "autoLogin": true,        // Автоматический логин
    "password": "qwer123Q"    // Пароль для /login
  }
}
```

## Важные замечания

⚠️ **Внимание:** Использование ботов может быть запрещено правилами некоторых серверов. Убедитесь, что у вас есть разрешение администрации на использование ботов на сервере DeltaWorld.

## Решение проблем

### Ошибка подключения

Если боты не могут подключиться:
1. Проверьте, что сервер онлайн
2. Попробуйте изменить `host` на `166.1.144.20` в config.json
3. Убедитесь, что указана правильная версия Minecraft

### Боты выкидываются с сервера

- Проверьте, разрешены ли боты на сервере
- Уменьшите количество ботов в config.json
- Увеличьте `spawnDelay` для более медленного подключения

### Боты не используют команды

- Убедитесь, что на сервере доступны команды `/rtp`, `/kit start` и т.д.
- Проверьте права доступа к этим командам
