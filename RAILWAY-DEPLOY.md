## 🚀 Деплой за 5 минут

### Шаг 1: Подготовьте GitHub репозиторий

```bash
# Инициализируйте Git (если еще не сделали)
git init
git add .
git commit -m "Railway deploy"

# Создайте репозиторий на GitHub
# Затем:
git remote add origin https://github.com/ВАШ_ЛОГИН/minecraft-bots.git
git branch -M main
git push -u origin main
```

### Шаг 2: Создайте проект на Railway

1. Откройте https://railway.app
2. Нажмите "Login" и войдите через GitHub
3. Нажмите "New Project"
4. Выберите "Deploy from GitHub repo"
5. Выберите ваш репозиторий `minecraft-bots`

### Шаг 3: Настройте переменные окружения

В Railway Dashboard → Settings → Variables добавьте:

```env
MC_HOST=mc.deltaworld.go-mc.pro
MC_PORT=25565
MC_VERSION=1.16.5
MC_PASSWORD=qwer123Q
BOT_COUNT=5
AUTO_START=true
PORT=3000
```

### Шаг 4: Деплой!

Railway автоматически:
- Обнаружит Node.js проект
- Установит зависимости
- Запустит `npm start`
- Выделит публичный URL

### Шаг 5: Откройте веб-панель

1. В Railway Dashboard найдите "Settings" → "Networking"
2. Нажмите "Generate Domain"
3. Ваш URL: `https://ваш-проект.up.railway.app`

## 🆘 Решение проблем

### Проблема: Боты не подключаются
**Решение:** 
- Проверьте логи в Railway Dashboard
- Проверьте переменные окружения
- Убедитесь, что `MC_HOST` правильный

### Проблема: Приложение засыпает
**Решение:**
- Railway не имеет автосна
- Боты работают постоянно
- Если нет трафика - все равно работает

### Проблема: Превышен лимит $5
**Решение:**
- Добавьте платежный метод
- Или создайте новый аккаунт
- Или используйте VPS
