# 🎲 Случайные ники из data.yml

Боты теперь могут использовать случайные реальные никнеймы из файла data.yml!

## 🚀 Как это работает:

### 1️⃣ **Автоматическая загрузка**
При запуске ботов система автоматически:
- Читает файл `data.yml`
- Загружает все ники из секции `collected-names`
- Показывает количество загруженных ников

### 2️⃣ **Случайный выбор**
Каждому боту присваивается:
- Случайный ник из списка (если `useRandomNames: true`)
- Или стандартный префикс + номер (если `useRandomNames: false`)

### 3️⃣ **Реалистичность**
Боты выглядят как настоящие игроки с именами:
- `Vanya_v_Tapkax`, `Bill_Ding`, `lizzard`
- `ChcoXD`, `_ham00001_`, `ZloyT3ry`
- И еще 2000+ вариантов!

## ⚙️ Настройка

### В config.json:

```json
{
  "bots": {
    "count": 5,
    "namePrefix": "Bot",
    "useRandomNames": true  // ← Включить случайные ники
  }
}
```

### Через веб-панель:

1. Откройте http://localhost:3000
2. Найдите чекбокс **"Случайные ники из data.yml"**
3. Поставьте галочку ✅
4. Нажмите **"Сохранить настройки"**
5. Запустите ботов

## 📊 Статистика:

В файле `data.yml` доступно:
- **2000+ уникальных никнеймов**
- Реальные ники игроков
- Разные стили: русские, английские, цифры
- Без повторений при запуске

## 🎯 Примеры использования:

### Пример 1: Все боты со случайными никами
```json
{
  "bots": {
    "count": 10,
    "useRandomNames": true
  }
}
```
Результат: `Vanya_v_Tapkax`, `lizzard`, `ChcoXD`, `ZloyT3ry`, etc.

### Пример 2: Стандартные префиксы
```json
{
  "bots": {
    "count": 10,
    "namePrefix": "Player",
    "useRandomNames": false
  }
}
```
Результат: `Player1`, `Player2`, `Player3`, etc.

## 🎮 Для каких файлов работает:

✅ **index.js** - множественные боты (npm start)
✅ **server.js** - боты через веб-панель (npm run web)
❌ **single-bot.js** - одиночный бот (требует доработки)
❌ **advanced-bot.js** - умный бот (требует доработки)

## 💡 Преимущества:

1. **Реалистичность** - боты выглядят как настоящие игроки
2. **Разнообразие** - 2000+ уникальных имен
3. **Настраиваемость** - легко включить/выключить
4. **Автоматизация** - не нужно придумывать имена

## ⚠️ Важно:

- Имена выбираются **случайно** при каждом запуске
- Могут быть **повторения** если ботов больше чем имен
- Имена берутся из **data.yml** без изменений
- Если файл не найден, используется **стандартный префикс**

## 🔧 Добавление своих ников:

Отредактируйте `data.yml`:

```yaml
collected-names:
  - МойНик1
  - МойНик2
  - МойНик3
  # Добавьте сколько угодно
```

Перезапустите ботов - новые ники будут использоваться!

## 📊 Проверка загрузки:

При запуске ботов в консоли должно появиться:
```
✓ Загружено 2000 никнеймов из data.yml
```

Если видите ошибку:
```
✗ Ошибка загрузки data.yml: ...
```

Проверьте:
1. Файл `data.yml` существует в корне проекта
2. Файл имеет правильный формат YAML
3. Установлен модуль `js-yaml` (npm install)
