# 🚀 Развертывание на Vercel

## ⚠️ Важное ограничение Vercel

**Vercel не поддерживает долгие WebSocket соединения!**

Minecraft боты требуют постоянного подключения к серверу, что невозможно на Vercel из-за:
- Таймаут выполнения функций (10 сек на Hobby, 60 сек на Pro)
- Serverless архитектура (функции завершаются после ответа)
- Отсутствие постоянных процессов

## 💡 Рекомендуемое решение

### Архитектура "Клиент + Облачная панель":

```
┌─────────────────┐         ┌──────────────────┐
│  Ваш компьютер  │         │   Vercel Cloud   │
│                 │         │                  │
│  bot-client.js  │◄───────►│   Веб-панель     │
│   (Боты тут)    │   API   │  (Управление)    │
└─────────────────┘         └──────────────────┘
        │
        ▼
┌─────────────────┐
│ Minecraft Сервер│
│  DeltaWorld     │
└─────────────────┘
```

## 🎯 Два варианта использования:

### Вариант 1: Только локально (Проще)

```bash
# Запуск клиента с ботами
npm run client
```

**Преимущества:**
- ✅ Все работает локально
- ✅ Не нужен интернет для панели
- ✅ Простая настройка

**Недостатки:**
- ❌ Нет веб-интерфейса
- ❌ Управление через консоль

### Вариант 2: Альтернативные хостинги

Используйте платформы с поддержкой долгих процессов:

#### 🟢 **Railway.app** (Рекомендуется)
```bash
# 1. Установите Railway CLI
npm i -g @railway/cli

# 2. Логин
railway login

# 3. Инициализация
railway init

# 4. Деплой
railway up
```

**Особенности:**
- ✅ Поддерживает долгие процессы
- ✅ $5/месяц бесплатно
- ✅ Автодеплой из GitHub
- ✅ Веб-панель работает

#### 🔵 **Render.com**
```bash
# 1. Создайте репозиторий на GitHub
# 2. Подключите к Render.com
# 3. Выберите "Web Service"
# 4. Build: npm install
# 5. Start: node server.js
```

**Особенности:**
- ✅ 750 часов бесплатно/месяц
- ✅ Простой деплой
- ✅ SSL сертификат
- ⚠️ Засыпает после неактивности

#### 🟣 **Heroku**
```bash
# 1. Установите Heroku CLI
# 2. Логин
heroku login

# 3. Создание приложения
heroku create my-mc-bots

# 4. Деплой
git push heroku main
```

**Особенности:**
- ✅ Известная платформа
- ✅ Много документации
- ❌ Платный ($7/месяц)

#### 🟠 **DigitalOcean App Platform**
```bash
# Через веб-интерфейс:
# 1. Создайте аккаунт
# 2. App Platform → Create App
# 3. Подключите GitHub репозиторий
# 4. Деплой
```

**Особенности:**
- ✅ $5/месяц
- ✅ Стабильный
- ✅ SSH доступ

## 📦 Подготовка к деплою

### 1. Создайте `.env` файл:

```env
MC_HOST=mc.deltaworld.go-mc.pro
MC_PORT=25565
MC_VERSION=1.16.5
MC_PASSWORD=qwer123Q
AUTO_START=true
PORT=3000
```

### 2. Обновите `package.json`:

```json
{
  "scripts": {
    "start": "node bot-client.js",
    "web": "node server.js"
  }
}
```

### 3. Создайте `Procfile` (для Heroku):

```
web: node server.js
worker: node bot-client.js
```

## 🔧 Конфигурация для Railway

Создайте `railway.json`:

```json
{
  "build": {
    "builder": "NIXPACKS"
  },
  "deploy": {
    "startCommand": "node server.js",
    "restartPolicyType": "ON_FAILURE",
    "restartPolicyMaxRetries": 10
  }
}
```

## 🆘 Проблемы и решения:

### Проблема: Боты не подключаются
**Решение:** Проверьте версию Minecraft в config.json

### Проблема: Таймаут на Vercel
**Решение:** Используйте Railway или Render

### Проблема: Боты отключаются через 5 минут
**Решение:** Анти-AFK уже встроен, проверьте настройки сервера
