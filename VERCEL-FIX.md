# 🔧 Исправление 404 на Vercel

## Проблема решена! ✅

Ошибка **404: NOT_FOUND** возникала потому что:
1. Vercel не может запускать долгие процессы (боты)
2. Неправильная конфигурация маршрутов

## 🎯 Новое решение

Создана **статическая веб-панель** для настройки ботов:
- Работает на Vercel ✅
- Только фронтенд (без серверной части) ✅
- Экспорт/импорт конфигурации ✅
- Боты запускаются локально на вашем ПК ✅

## 🚀 Деплой на Vercel

### 1. Загрузите проект на GitHub
```bash
git init
git add .
git commit -m "Minecraft Bots Web Panel"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГИН/minecraft-bots.git
git push -u origin main
```

### 2. Импортируйте в Vercel
1. Откройте https://vercel.com
2. Нажмите "Add New" → "Project"
3. Импортируйте ваш GitHub репозиторий
4. Нажмите "Deploy"

### 3. Готово!
Ваша панель будет доступна по адресу:
```
https://ваш-проект.vercel.app
```

## 💻 Использование

### На Vercel (веб-панель):
1. Откройте ваш URL на Vercel
2. Настройте параметры ботов
3. Экспортируйте config.json
4. Скачайте файл

### На вашем ПК (боты):
1. Скопируйте config.json в папку проекта
2. Запустите:
```bash
npm install
node bot-client.js
```

## ⚙️ Настройка vercel.json

Файл уже настроен правильно:
```json
{
  "version": 2,
  "rewrites": [
    {
      "source": "/",
      "destination": "/public/standalone.html"
    },
    {
      "source": "/(.*)",
      "destination": "/public/$1"
    }
  ]
}
```

## 🆘 Проблемы и решения

### Проблема: 404 на Vercel
**Решение:** Пересоздайте проект с новым vercel.json

### Проблема: Боты не запускаются
**Решение:** Боты должны работать локально, не на Vercel

### Проблема: Не скачивается config.json
**Решение:** Проверьте разрешения браузера на скачивание
