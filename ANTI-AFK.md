# 🛡️ Защита от AFK (Анти-кик)

Все боты теперь защищены от выкидывания за неактивность!

## 🔄 Как это работает?

### Для всех типов ботов:

#### 1️⃣ **Глобальная защита (каждые 10 секунд)**
- Поворот головы в случайную сторону
- Случайные прыжки (20% вероятность)
- Отправляется как движение в сервер

#### 2️⃣ **Специфичная защита для каждого типа:**

### 🏃 **Бот-Бегун:**
- Активно бегает каждые 1.5-3.5 секунды
- Дополнительные повороты головой каждые 8 секунд
- Спринт и прыжки
- **Самый защищенный** от AFK

### 🎮 **Бот-Командир:**
- Использует команды каждые 30-90 секунд
- Повороты головой каждые 12 секунд
- Случайные прыжки
- Команды считаются активностью

### 💬 **Бот-Болтун:**
- Пишет в чат каждые 45-120 секунд
- Повороты головой каждые 15 секунд
- Случайные прыжки
- Сообщения в чате = активность

## ⏱️ Таймлайн активности:

```
0 сек    - Бот заходит
1 сек    - /login qwer123Q
2.5 сек  - /kit start
4 сек    - /rtp safe
6 сек    - Начинает поведение
8-15 сек - Первое анти-AFK действие
```

## 🎯 Типы анти-AFK действий:

1. **bot.look()** - Поворот головы (самое легкое)
2. **bot.setControlState('jump')** - Прыжок
3. **bot.chat()** - Сообщения в чат
4. **bot.setControlState('forward')** - Движение

## 📊 Частота действий:

| Тип бота    | Основные действия | Анти-AFK    | Защита     |
|-------------|-------------------|-------------|------------|
| Бегун       | 1.5-3.5 сек       | 8 сек       | ⭐⭐⭐⭐⭐ |
| Командир    | 30-90 сек         | 12 сек      | ⭐⭐⭐⭐   |
| Болтун      | 45-120 сек        | 15 сек      | ⭐⭐⭐⭐   |

## ✅ Что предотвращает:

- ❌ Выкидывание за AFK (Away From Keyboard)
- ❌ Таймауты от неактивности
- ❌ Автоматические кики через 5 минут
- ❌ Дисконнект от сервера

## 🔧 Технические детали:

### Интервалы в коде:

```javascript
// Глобальная защита (все боты)
setInterval(() => {
    bot.look(random_yaw, 0, false);
    if (random > 0.8) bot.jump();
}, 10000);

// Бегун - дополнительная защита
setInterval(() => {
    bot.look(random_yaw, 0, false);
}, 8000);

// Командир - защита
setInterval(() => {
    bot.look(random_yaw, 0, false);
    if (random > 0.7) bot.jump();
}, 12000);

// Болтун - защита
setInterval(() => {
    bot.look(random_yaw, 0, false);
    if (random > 0.8) bot.jump();
}, 15000);
```

## 💡 Советы:

1. **Не беспокойтесь** - боты будут работать 24/7
2. **Бегуны самые надежные** - они постоянно двигаются
3. **Болтуны и Командиры** - тоже защищены минимальными движениями
4. **Все типы работают вместе** - можете запускать разные типы

## 🔍 Проверка работы:

Если бот всё равно выходит:
1. Проверьте логи - там будет причина
2. Возможно сервер имеет другие правила
3. Попробуйте увеличить частоту движений
4. Используйте больше ботов-бегунов

## ⚠️ Важно:

Анти-AFK работает только когда:
- ✅ Бот подключен к серверу
- ✅ Бот успешно заспавнился
- ✅ `bot.entity` существует

Если бот еще не заспавнился, защита не сработает!
